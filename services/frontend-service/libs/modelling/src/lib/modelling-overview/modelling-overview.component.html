Modelling
Manually Only text input

Plannning models can be entered via textual input in a Web IDE. Several modeling languages are supported (e.g. PDDL, SHOP, HDDL).

Interface Graphical modelling

The modeling of the planning domain and problems is supported by graphical input. No modeling code needs to be written.

Automated Sensor-based modelling

In this mode you only need to create a configuration that automates the creation of models. This option is particularly suitable for IoT and cyber-physical systems.
